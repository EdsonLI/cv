# Edson Luís Isele — Currículo digital

Currículo profissional responsivo desenvolvido como site estático para GitHub Pages.

## Estrutura

- `index.html`: conteúdo e semântica da página
- `assets/css/styles.css`: design system e responsividade
- `assets/js/main.js`: navegação, tema, animações e progresso de leitura
- `assets/files/Curriculo_Edson_Luis_Isele.pdf`: versão PDF para download
- `.github/workflows/deploy-pages.yml`: publicação automática no GitHub Pages

## Publicação

Todo push na branch `main` dispara o GitHub Actions e publica o site em `https://edsonli.github.io/cv/`.
